这篇文章我会通过 Q&A 的形式简单介绍一下我自己。

## 我的经历

很多老读者应该比较清楚，我是 18 年北京理工大学软件工程专业本科毕业的，刚毕业就去了某家互联网公司。

我的籍贯是内蒙古东北某地人，高考成绩640分，我大学毕业后就留在了北京工作，在大学期间获得了多项奖学金。不过，还好我没有因为学校而放松自己，反倒在学校犹如像高中一样比身边的同学都要更努力，整个大学还算过的比较充实。

## 我坚持写了多久博客？

我自己是从大一就开始写博客了。那时候就是随意地在CSDN博客平台上发发自己的学习笔记和自己写的代码。

身边也有很多小伙伴经常问我：“我现在写博客还晚吗？”

如果你想做什么事情，尽量少问迟不迟，多问自己值不值得，只要你觉得有意义，就尽快开始做吧！人生很奇妙，我们每一步的重大决定，都会对自己未来的人生轨迹产生影响。是好还是坏，也只有我们自己知道了！

对我自己来说，坚持写博客这一项决定对我人生轨迹产生的影响是非常正面的！所以，我也推荐大家养成坚持写博客的习惯。

## 我在大学期间赚了多少钱？

在校期间，我还通过办培训班、接私活、技术培训等方式变现 10W多，成功实现了“经济独立”。近几年我用自己赚的钱买了一辆特斯拉，大概40W+，还给家里补贴了很多，减轻了父母的负担。

其实，我在大学就这么努力地开始赚钱，也主要是因为家庭条件太一般，父母赚钱都太辛苦了！也正是因为我自己迫切地想要减轻父母的负担，所以才会去尝试这么多赚钱的方法。

我发现做咱们程序员这行的，很多人的家庭条件都挺一般的，选择这个行业的很大原因不是因为自己喜欢，而是为了多赚点钱。

如果你也想通过接私活变现的话，可以在我的公众号后台回复“**接私活**”来了解一些我的个人经验分享。

生活本就是有苦有甜。共勉！
